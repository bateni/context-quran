\startbuffer[\q:29:55]

\stopbuffer
\startbuffer[\q:29:56]
یَٰعِبَادِیَ ٱلَّذِینَ ءَامَنُوۤا۟ إِنَّ أَرۡضِی وَٰسِعَةࣱ فَإِیَّٰیَ فَٱعۡبُدُونِ%
\stopbuffer
\startbuffer[\q:29:57]
كُلُّ نَفۡسࣲ ذَاۤئِقَةُ ٱلۡمَوۡتِۖ ثُمَّ إِلَیۡنَا تُرۡجَعُونَ%
\stopbuffer
\startbuffer[\q:29:58]
وَٱلَّذِینَ ءَامَنُوا۟ وَعَمِلُوا۟ ٱلصَّٰلِحَٰتِ لَنُبَوِّئَنَّهُم مِّنَ ٱلۡجَنَّةِ غُرَفࣰا تَجۡرِی مِن تَحۡتِهَا ٱلۡأَنۡهَٰرُ خَٰلِدِینَ فِیهَاۚ نِعۡمَ أَجۡرُ ٱلۡعَٰمِلِینَ%
\stopbuffer
\startbuffer[\q:29:59]

\stopbuffer
\startbuffer[\q:29:60]
وَكَأَیِّن مِّن دَاۤبَّةࣲ لَّا تَحۡمِلُ رِزۡقَهَا ٱللَّهُ یَرۡزُقُهَا وَإِیَّاكُمۡۚ وَهُوَ ٱلسَّمِیعُ ٱلۡعَلِیمُ%
